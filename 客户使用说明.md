# 工厂 AI 视频工厂 使用说明

## 首次使用

1. 双击 `install_factory_console.command`
2. 安装完成后，双击 `start_factory_console.command`
3. 浏览器打开 `http://127.0.0.1:3001`

## 日常启动

1. 双击 `start_factory_console.command`
2. 打开浏览器访问 `http://127.0.0.1:3001`

## 日常关闭

1. 双击 `stop_factory_console.command`

## 页面说明

- `总览面板`
  查看素材数量、视频任务数量和最近任务
- `资料素材库`
  查看与同步产品图、工厂图、案例图、数字人、视频样片和文档
- `视频工厂`
  填写产品名称、视频标题、适用对象、视频重点、素材约束，生成脚本和视频
- `成片中心`
  查看已生成成片、封面文案、标题和标签，完成审核
- `任务中心`
  查看当前任务和操作记录

## 常用流程

1. 先进入 `资料素材库`，确认素材已经同步
2. 进入 `视频工厂`，填写视频需求
3. 点击 `生成脚本`
4. 检查脚本后点击 `生成视频`
5. 在 `成片中心` 查看成片并审核

## 默认地址

- 前端：`http://127.0.0.1:3001`
- 后端：`http://127.0.0.1:8001`

## 注意事项

- 素材和数据默认保存在本机
- 如更换电脑，需要重新安装依赖
- 如需接入视频生成 API，由客户自行提供 API 账号和额度
