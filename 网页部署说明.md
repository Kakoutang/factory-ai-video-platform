# 工厂 AI 视频工厂 网页部署说明

## 目标

部署完成后，客户只需要打开一个网址即可使用，不需要安装本地依赖。

## 部署结构

- `web`
  对外提供网页入口
- `frontend`
  客户工作台前端
- `backend`
  接口、数据库、素材同步

## 部署前准备

1. 准备一台云服务器
2. 安装 Docker 和 Docker Compose
3. 准备一个域名并解析到服务器

## 需要放到服务器上的内容

把整个项目目录上传到服务器：

- `ai-growth-platform`

重点目录：

- `deploy/data/client-assets`
  这里放客户的产品图、工厂图、案例图、数字人、视频样片和文档
- `deploy/nginx/default.conf`
  网页入口配置
- `docker-compose.web.yml`
  网页部署配置

## 首次部署

```bash
cd ai-growth-platform
cp .env.web.example .env.web
docker compose --env-file .env.web -f docker-compose.web.yml up -d --build
```

## 更新素材

把新素材放进：

```bash
deploy/data/client-assets
```

然后在网页里进入 `资料素材库` 点击 `同步素材库`。

## 更新版本

```bash
cd ai-growth-platform
docker compose --env-file .env.web -f docker-compose.web.yml up -d --build
```

## 默认访问方式

- 服务器 IP：`http://你的服务器IP`
- 域名：`http://你的域名`

如需 HTTPS，建议把域名接到现有网关或再加一层 HTTPS 代理。

## 数据保存位置

- 数据库：`deploy/data/app.db`
- 素材目录：`deploy/data/client-assets`

## 适合交付给客户的方式

1. 你自己部署到服务器
2. 给客户一个网址
3. 客户只负责登录和使用

这样客户不需要安装任何东西。
